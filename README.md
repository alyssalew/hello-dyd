#Hello DYD?
This application creates a webpage that monitors the Donate Your Data services and send an email when a service is down.

##How to run app on your local machine:
1. Clone repository to your local machine from here: https://github.com/alyssalew/hello-dyd.git
2. Navigate to the directory where you cloned the repository and run the command:
'$ node app.js'
3. Visit localhost:3000 in your favorite web browser to see the status of the services

###To receive email alerts from the application:
1. Open the file “app.js”
2. Add your email to the list of “recipients”
